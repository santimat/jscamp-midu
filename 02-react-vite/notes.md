# React with vite

- Vite es un empaquetador que nos permite convertir react para que pueda ser interpretado por el navegador
- Crear react app con vite:
  1. pnpm create vite@latest
  2. ponemos el nombre del proyecto
  3. elegimos el framework
  4. elegimos el lenguaje, en este caso (js + SWC)
  5. nos va a preguntar por rolldown-vite (si sigue experimental decimos n)
  6. instalmos dependencias
